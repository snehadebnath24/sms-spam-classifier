# 📩 SMS/Email Spam Classifier
📌 Overview

This project is a machine learning model that classifies messages as Spam or Not Spam (Ham). It helps in filtering unwanted messages and improving communication efficiency.

🚀 Features
Classifies SMS/Email as Spam or Ham
Text preprocessing (removing stopwords, punctuation, etc.)
Uses Machine Learning algorithms
Simple and user-friendly interface

🛠️ Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Natural Language Processing (NLP)
- Streamlit
